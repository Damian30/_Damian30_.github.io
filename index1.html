Project Summary

The objective of the project was to answer a funadamental question: "What is the weather like as we approach the equator". Also, to visualize the weahter of 500+ cities accross the world, varying distance of the equator. We use random coordiantes in order to select our cities by creating multiple API calls via Python program.

Once we gather all the relevant data, we created multiple plots using "Matplotlib" to visualize Temparature, Cloudiness, Humidity and Wind speed.
